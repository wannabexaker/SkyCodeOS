# Agent Definition Standard (V1)

## Required Files

- soul.yaml
- heart.yaml
- mind.yaml
- doctrine.yaml

## Minimal Required Fields

### soul.yaml
- id
- name
- role
- core_values

### heart.yaml
- communication_style
- collaboration_style

### mind.yaml
- planning_depth
- risk_tolerance
- validation_style

### doctrine.yaml
- must_never
- must_always
- approval_required_for

## V1 Policy

- Keep definitions minimal and stable.
- Defer emotional drift and personality evolution mechanics.
